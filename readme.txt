== Description ==

CV Portal allows you to collect and manage cv from your website.


== Major features ==

* Custom post type.
* Can Download.
* You can collect cv.
* Custom Taxonomy.


== Installation ==

Upload the cv portal plugin to your website, activate it, and then copy the "[cv]" shortcode past it where you want to show output.

1, 2, 3: You're done!
